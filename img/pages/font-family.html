Belajar CSS: font-family

HTML dan CSS merupakan bagian tak terpisahkan dari website modern saat ini. HTML digunakan untuk membuat konten atau kerangka logis dari halaman web, sedang CSS digunakan untuk mengatur tampilan dari website, seperti warna dan font yang digunakan.

CSS biasanya selalu dikaitkan dengan HTML, karena keduanya memang saling melengkapi. HTML ditunjukan untuk membuat struktur, atau konten dari halaman web. Sedangkan CSS digunakan untuk tampilan dari halaman web tersebut. Istirahatnya, HTML for content, CSS for Presentation
